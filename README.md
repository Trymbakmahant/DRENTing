## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd client
  cd server
```

Install dependencies

```bash
  do in boath client and server "npm install" 
```

Start the server
&
start the client 
```bash
  npm start
```
